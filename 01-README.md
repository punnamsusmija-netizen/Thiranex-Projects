## 🛠️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Thiranex-Projects.git
```

Navigate to the project folder:

```bash
cd Thiranex-Projects
```

Install required dependencies:

```bash
pip install pandas numpy matplotlib seaborn
```

## 🚀 Usage

Run the Python scripts or Jupyter notebooks included in each project directory:

```bash
python weather_analysis.py
```

or

```bash
jupyter notebook
```
